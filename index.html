The Capstone Project: HabitTrack Pro

Our entire course revolves around building one application from scratch: a professional-grade, beautifully designed habit tracker. This project is meticulously crafted to teach you the most in-demand skills and give you a portfolio piece that stands out.

Your 5-Week Transformation Plan

Week 1: Foundations & Data Layer

We start with a professional setup: Android Studio, Git, and Gradle. Then, we build and test the complete database layer using Room (Entity, DAO, Type Converters) and write our first instrumented tests.

Week 2: Architecture & Business Logic

Implement the Repository pattern following Clean Architecture principles. Master Dependency Injection with Hilt, then build and unit test the ViewModel, learning modern state management with StateFlow.

Week 3: UI with Jetpack Compose

A deep dive into Jetpack Compose. We'll build reusable UI components and create a beautiful, dynamic Home screen using `LazyColumn` and `Scaffold`, connecting the UI to the ViewModel to display real data.

Week 4: Interaction & Navigation

Bring the app to life by handling UI events and gestures. We'll implement multi-screen navigation with Compose Navigation and build the "Add/Edit Habit" screen, passing data and creating new habits.

Week 5: Advanced Topics & Career Prep

Implement Firebase Authentication, add UI polish, and conduct UI testing with Espresso. Finally, we'll prepare a release-ready App Bundle and learn how to showcase your project on your resume and GitHub.
